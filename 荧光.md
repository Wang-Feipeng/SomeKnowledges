# 颜色

* 颜色*是光和眼睛相互作用而产生的，（色即是空？）*是由人的视觉得到的，因此只有在光照情况下，物质的颜色才能为肉眼所见，如果在没有光线的密闭暗室中，*物体的颜色是看不见的* （没有光，物体的颜色？）。

* 光具有波粒二象性。在整个电磁波谱中，可见光波长范围大约为400-800nm，光的波长不同，就会引起不同的视觉，即感觉到不同的颜色。

* 只有一种波长的光称为单色光，由具有不同波长的单色光组成的光称为复色光。若将两种色光按一定比例混合而得到白光，这两种颜色就成为互补色。如蓝光和黄光。若果在白光中除去一种补色，则可以观察到另一种补色。如让白光通过滤色片除去蓝绿光，眼睛观察到的将是红光。这种从白光中除去部分色光得到另一种色光的过程称为减色混合。两种或多种色光混合，可以得到另一种色光。如黄光可由红光和绿光合成。自然界中很少有纯的单色光，我们接触到的大多数颜色都是通过减色混合过程产生的。

* 物质之所以呈现出某种颜色，一般是由于物质有选择地吸收了白光中的某种波长的光，从而呈现出与之互补的那种光的颜色。如硫酸铜因吸收白光中的黄光而呈现黄色，高碘酸钾因吸收白光中的绿色而呈现紫色。如果白光照到物体上无任何色光被吸收，我们看见白色，反正，如果入射光全被吸收，则物质为黑色。

* 物质呈现不同颜色是由于对不同波长的光吸收（波动性）、反射（粒子型）程度不同。这与组成该物质的分子、离子的内部结构有关系。物质是由原子组成，而原子又是由原子核和电子组成。原子有许多能量不同但有确定值的状态，电子可以从一种状态跃迁到另一种状态，在跃迁过程中，同时要吸收一定的能量或释放出一定的能量。这一能量可以以光的形式提供吸收或辐射放出。不仅原子，物质的分子或离子也有这种类似的确定的能量状态，分子中电子可在不同状态将跃迁，引起对光的吸收或辐射。由于各种物质分子的能量状态不同，因为对可见光中不同波长的光吸收便不同，这种差异，直接决定这物质的颜色。简单地说，物质之所以能呈现各种不同的颜色，就是因为物质在光源（太阳光或其他灯光）提供的能量作用下，构成物质的分子或原子中电子选择性吸收一定波长的光从低能量跃迁到高能量状态，或由某一高能量状态跃迁回低能量状态，并发射出特定波长的光，从而显示其特有的颜色。

* 光之所以选择性吸收光子，主要涉及一个能量匹配的问题。因为物质分子或原子中电子能量状态的能量是一个确定值，因此在两个不同状态发生跃迁，需要的能量值就是这两个状态能量值的差值（E1-E2）；另一方面，一定波长的光具有一定的能量（E=hc/r），要发生跃迁，就要负荷吸收的能量等于发生跃迁所需的能量(E1-E2=hc/r)。由于E1、E2、h和c固定，因此r也只能是某个数值。这中间，由于能量状态的复杂性，试试上选择性吸收或放出的光波长并不只是某个数值，而是一个狭窄的范围。

* 除了主要取决于分子或离职的电子层结构外，还有许多其他多种因素如物质聚集状态、温度都有关系。


# 荧光物质

* 有机荧光物质能吸收特定频率的光，并发射出低频率（较长波长）的荧光释放所吸收的能量。

* 有色化合物分子通常处于能量最低的状态，称为基态。吸收紫外或可见光的能量后，电子跃迁至高能量轨道激发态。分子可有多个激发态。处于激发态的分子通过振动驰豫、内部转换等过程跃迁到分子的最低激发态的最低振动能级，再发生辐射跃迁回到基态，放出光子，产生荧光。**荧光是冷光，颜色为蓝、绿、红和黄**

* 有机染料分子的第一激发态与基态的能差是一定的，因而**荧光波长不随激发光波长的改变而发生改变**。分子激发过程中吸收的能量一般高于荧光辐射释放的能量，二者之差以热的形式损耗，因此荧光波长比激发波长长，其差通常为50-70nm，当有机化合物内可形成氢键时，则增至150-250nm，这一规律称为**stokes位移**。荧光的强度受许多因素的制约，如激发光源能量、吸收强度、量子效率等。量子效率也称量子收率，是指荧光物体分子发射的光量子数与吸收的光量子数之比。其大小是由分子结构决定的，而与激发光源的能量无关。事实证明，荧光物质分子一般都含有发射荧光的基团称为（荧光团）以及能使吸收波长改变并伴随荧光增强的助色团。

> 分子结构与荧光特性

* 分子产生荧光必须具备两个条件：具有合适的结构；具有一定的荧光量子产率。

* 化合物的结构与荧光

>> 1. 共轭效应：提高共轭度有利于增加荧光效率并产生红移。

>>> 芳环越大，荧光峰越移向长波方向

>>> 同一共轭环数的芳族化合物，线性环结构者荧光波长比非线性者要强。

>> 2. 刚性平面结构：试剂本身有刚性平面结构；形成络合物后，形成刚性平面；取代基之间形成氢键，加强分子刚性结构和增强荧光强度；异构体的影响，顺式和反式同分异构体具有不同的荧光强度。

>>> 可降低分子振动，减少与溶剂的相互作用，故具有很强的荧光。

>> 3. 取代基效应

>>> 芳环上有供电基，使荧光增强：给电子基团常使荧光增强；吸电子基团会减弱甚至破坏荧光；邻、对位取代基增强荧光，间位取代抑制荧光；重原子引入体系，荧光强度都很弱，而磷光强度相应增强；与π电子体系作用小的取代基，影响小。

>> 4. 跃迁类型：π\*→π的荧光效率高，系间窜越过程的速率常数小，有利于荧光的产生。


1. 共轭系统对荧光的影响

* 通常增加分子π电子共轭体系长度可提高荧光效率并使荧光红移。空间位阻效应的存在能够破坏分子的共平面性及共轭程度，从而使荧光减弱。立体异构对荧光强度也有影响，如反式二苯乙烯是强荧光型的，顺式二苯乙烯由于位阻效应的存在则无荧光特性。

2. 取代基对荧光的影响

大部分有机荧光物质分子中带有芳环，芳环上引入取代基可改变荧光的光量子收率和发射波长。通常邻、对位定位基可使荧光增强，间位定位基使荧光减弱，硝基、偶氮基能阻止荧光的产生。分子两端分别引入给电性和吸电性基团可使染料发生红移并伴随荧光的增强。卤原子的存在对荧光不利。氨基的引入可使荧光增强。

3. 分子环构化对荧光的影响

染料分子的闭环对荧光的产生非常有利，可以增加分子共平面性和刚性而使荧光增强。许多本身无荧光或荧光很弱的化合物与金属鳌合产生的具有环状结构的螯合物显示较强荧光。分子内含有羟基并可形成分子内氢键多数情况下能使荧光强度提高。

4. 荧光与受激拉曼散射

拉曼散射光谱是研究其物质结构的强有力工具。拉曼散射光谱是指分子对入射光所产生使其频率发生较大改变的一种光散射现象。激光拉曼光谱主要的一些特点包括：

> 每种物质（分子）都有自己完全独立的特征谱线，因此每种物质的特征谱线可以表征这一物质。

> 拉曼谱线的线宽大多数较窄，并且往往都是成对出现的，也就是具有完全相同大小的正负频差。这两条谱线在短波一遍的叫做反斯托克斯谱线，在长波一边的叫做斯托克斯谱线。

> 每一物质的拉曼频移（也就是入射频率与散射频率之差）的大小和入射光的频率是完全无关的，拉曼散射是瞬间产生的，即入射光消失时，拉曼散射也会在瞬间消失这个时间大约会在10

> 拉曼频率位移会存在一个很大的范围内，它有可能会使几个波数（cm/1），也有可能会达到3800个波数。

> 不同谱线的拉曼谱线强度和偏振性是完全不一样的。

> 可以用斯托克斯和反斯托克斯的强度之比来表示分子在基态的时候与在第一激发态的时候得布居数之比，这个比值是完全符合玻尔兹曼分布定律的，它同时也是热力学的温度函数，并且这个比值还与拉曼的位移有关。

> 在一切分子中也就是无论固体、液体还是气体，拉曼散射效应是大都是普遍存在的。

* 拉曼效应与荧光效应的关系

> 荧光效应与拉曼效应产生机理是完全不一样的。物质分子在吸收光量子之后产生了荧光效应，这个过程是从基态跃迁到振动激发态的过程，受激发的分子或者原词重新回到基态时，他们从外界获得的能量以荧光的形式重新发射出来。这与拉曼散射是完全不一样的。拉曼效应是让基态的分子在吸收入射光子的能量hv0后就会跃迁到一个特定的激发态hv1+hv0，而散射物质的分子中是没有这种激发态的。当分子发生跃迁从这种特定的激发态到振动的激发态时，分子会释放出能量hv0-h△v，这就是斯托克斯线的由来。同样的而远离当分子吸收入射的光子的时候，也同样会跃迁的一个特定的激发态hv2+h△v0，当这个分子发生跃迁从激发态到振动基态的时候，分子会释放出能量hv0+h△v；，这就是反斯托克斯线形成的原理。

> 荧光光谱可能被分子或原子所能吸收的一些频率来进行激发，而所有的频率都可以激发拉曼光谱。

* 瑞利散射光拉曼光

> 激发光的能量不足，将电子激发至基态中较高的振动能级，加入电子在受激后能量没有损失，并且在瞬时返回原来的能级，于是便在各个不同的方向发射和激发光相同波长的辐射，这种辐射称为瑞利散射光。

> 被激发到基态中其他较高振动能级的电子，当它返回到比原来的能级稍高或稍低时，，便伴随着产生波长略长或略短于激发光的拉曼散射光。

* 影响荧光强度的主要因素

1. 荧光猝灭

> 自猝灭

>> 荧光辐射的自吸收

>> 荧光物质的激发态分子M\*与基态分子M形成激发态的二聚体M*M

>> 基态的荧光物质分子的缔合

> 电荷转移猝灭

>> 激发态分子比基态具有更强的与其他物质发生氧化还原反应的能力，从而导致荧光猝灭。

> 转入三重态猝灭

>> 发射S1→T1间的系间窜越，把多余的能量消耗于碰撞之中使荧光猝灭。

2. 温度、酸度和溶剂的影响

> 温度：随着溶剂温度的降低，荧光物质溶液的荧光量子产率和荧光强度将增大。

> 酸度：对酸碱化合物，溶液pH的影响较大，需要严格控制。

> 溶剂：除一般溶剂效应外，溶剂的极性、氢键、配位键的形成都将使化合物的荧光发生改变。
